# 每日任务 App - UI设计规范

## 1. 设计原则

- **简洁**: 减少视觉噪音，突出核心内容
- **直观**: 操作路径短，用户一看就懂
- **一致**: 统一的组件样式和交互模式
- **舒适**: 色彩柔和，长时间使用不疲劳

## 3. 字体规范

| 类型 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 标题H1 | 24px | Bold | 页面主标题 |
| 标题H2 | 20px | SemiBold | 区域标题 |
| 标题H3 | 16px | SemiBold | 卡片标题 |
| 正文 | 14px | Regular | 正文内容 |
| 辅助文字 | 12px | Regular | 说明、时间等 |
| 按钮文字 | 16px | Medium | 按钮内文字 |

## 4. 间距系统（基于8px网格）

| 名称 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 极小间距 |
| sm | 8px | 小间距 |

| lg | 24px | 大间距 |
| xl | 32px | 超大间距 |

| 类型 | 数值 | 用途 |
|------|------|------|
| 小圆角 | 8px | 小按钮、标签 |
| 标准圆角 | 12px | 卡片、输入框 |

## 7. 交互动效

- **打卡动效**: 点击打卡按钮后，卡片背景短暂变绿，显示✓动画
- **升级动效**: 等级/段位提升时弹出庆祝动画
- **页面切换**: 使用Flutter默认的Material过渡动效
- **列表动画**: 任务列表使用淡入动画

## 8. 图标规范

- 使用Material Icons作为主要图标库
- 等级和段位使用Emoji图标
- 底部导航使用Material Icons
