# 每日任务 App - 数据库设计

## 数据库概述
- **数据库引擎**: SQLite (通过sqflite包)
- **数据库名称**: daily_task.db
- **版本**: 1

## 数据表设计

### 1. tasks 表（任务表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 任务ID |
| title | TEXT | NOT NULL | 任务名称 |
| description | TEXT | | 任务描述 |
| task_type | INTEGER | NOT NULL | 任务类型：1=每日重复，2=单日任务 |
| remind_time | TEXT | | 提醒时间，格式 HH:mm |
| remind_enabled | INTEGER | DEFAULT 0 | 是否开启提醒：0=关闭，1=开启 |
| repeat_days | TEXT | | 重复日，格式 "1,2,3,4,5"（周一到周五） |
| created_at | TEXT | NOT NULL | 创建时间 |
| is_active | INTEGER | DEFAULT 1 | 是否启用：0=删除，1=正常 |

**索引**:
- `idx_tasks_type` on (task_type)
- `idx_tasks_active` on (is_active)

### 2. check_ins 表（打卡记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 记录ID |
| task_id | INTEGER | FOREIGN KEY → tasks.id | 关联任务ID |
| check_in_date | TEXT | NOT NULL | 打卡日期，格式 YYYY-MM-DD |
| check_in_time | TEXT | NOT NULL | 打卡时间，格式 HH:mm:ss |
| created_at | TEXT | NOT NULL | 记录创建时间 |

**约束**:
- UNIQUE(task_id, check_in_date) — 同一任务同一天只能打卡一次

**索引**:
- `idx_checkins_task` on (task_id)
- `idx_checkins_date` on (check_in_date)

### 3. user_stats 表（用户统计数据表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 记录ID（固定为1，单用户） |
| total_check_ins | INTEGER | DEFAULT 0 | 累计打卡总次数 |
| current_streak | INTEGER | DEFAULT 0 | 当前连续打卡天数 |
| max_streak | INTEGER | DEFAULT 0 | 最长连续打卡天数 |
| last_check_in_date | TEXT | | 最后一次打卡日期 |
| level | INTEGER | DEFAULT 1 | 当前等级 |
| rank | INTEGER | DEFAULT 1 | 当前段位 |
| updated_at | TEXT | NOT NULL | 最后更新时间 |

### 4. focus_sessions 表（专注会话表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 会话ID |
| start_time | TEXT | NOT NULL | 开始时间 |
| end_time | TEXT | | 结束时间 |
| duration_minutes | INTEGER | DEFAULT 0 | 专注时长（分钟） |
| interruption_count | INTEGER | DEFAULT 0 | 被打断次数 |
| created_at | TEXT | NOT NULL | 记录创建时间 |

**索引**:
- `idx_focus_date` on (start_time)

### 5. settings 表（设置表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| key | TEXT | PRIMARY KEY | 设置键名 |
| value | TEXT | NOT NULL | 设置值 |

**预设设置项**:

| key | 说明 | 默认值 |
|-----|------|--------|
| theme_mode | 主题模式 | "system" (system/light/dark) |
| notification_enabled | 全局通知开关 | "true" |
| focus_ai_whitelist | 专注模式AI白名单 | "豆包,元宝,千问,ChatGPT,Claude,Gemini" |

## SQL建表语句

```sql
-- 任务表
CREATE TABLE tasks (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    description TEXT,
    task_type INTEGER NOT NULL,
    remind_time TEXT,
    remind_enabled INTEGER DEFAULT 0,
    repeat_days TEXT,
    created_at TEXT NOT NULL,
    is_active INTEGER DEFAULT 1
);

-- 打卡记录表
CREATE TABLE check_ins (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    task_id INTEGER NOT NULL,
    check_in_date TEXT NOT NULL,
    check_in_time TEXT NOT NULL,
    created_at TEXT NOT NULL,
    FOREIGN KEY (task_id) REFERENCES tasks(id),
    UNIQUE(task_id, check_in_date)
);

-- 用户统计表
CREATE TABLE user_stats (
    id INTEGER PRIMARY KEY DEFAULT 1,
    total_check_ins INTEGER DEFAULT 0,
    current_streak INTEGER DEFAULT 0,
    max_streak INTEGER DEFAULT 0,
    last_check_in_date TEXT,
    level INTEGER DEFAULT 1,
    rank INTEGER DEFAULT 1,
    updated_at TEXT NOT NULL
);

-- 专注会话表
CREATE TABLE focus_sessions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    start_time TEXT NOT NULL,
    end_time TEXT,
    duration_minutes INTEGER DEFAULT 0,
    interruption_count INTEGER DEFAULT 0,
    created_at TEXT NOT NULL
);

-- 设置表
CREATE TABLE settings (
    key TEXT PRIMARY KEY,
    value TEXT NOT NULL
);

-- 索引
CREATE INDEX idx_checkins_task ON check_ins(task_id);
CREATE INDEX idx_checkins_date ON check_ins(check_in_date);
CREATE INDEX idx_tasks_type ON tasks(task_type);
CREATE INDEX idx_tasks_active ON tasks(is_active);
CREATE INDEX idx_focus_date ON focus_sessions(start_time);
```

## 数据关系图

```
tasks (1) ──── (N) check_ins
   │
   └── 通过 task_id 关联

user_stats ──── 独立表（单用户，只有一行记录）

focus_sessions ──── 独立表（每次专注一条记录）

settings ──── 键值对配置表
